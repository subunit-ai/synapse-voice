Built for daily dictation

Alles drin. Nichts versteckt.

01

Hotkey-Diktat

Strg + Space druecken, sprechen, loslassen. Text landet sofort in der App, in der dein Cursor war. Keine Window-Switches.

02

99 Sprachen

Whisper-Stack: Deutsch, English, Francais, ... — alle 99 Sprachen die OpenAI Whisper unterstuetzt, mit suchbarer Auswahl.

03

AI-Cleanup

Optional: Aehs / Ums entfernen, Satzzeichen einsetzen, in Business-Tonalitaet umschreiben. Per Toggle pro Aufnahme.

04

Lexikon

Eigennamen + Marken die Whisper falsch versteht? Trag sie ein, wir biasen das Modell auf die richtige Schreibung.

05

Auto-Update

Neue Versionen werden automatisch geladen + installiert. SHA-256-verifiziert. Du klickst einmal "OK", Rest ist Auto-Pilot.

06

Open Source

Source auf GitHub. Pull-Requests willkommen. Kein Vendor-Lock-in. Deine Daten + dein Code, voll inspizierbar.
